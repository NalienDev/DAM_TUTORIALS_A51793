# 03 Screens

## 1. Main Screen (Activity)
- **Top Bar**: App title and Refresh icon.
- **Content**: A vertical RecyclerView showing a list/grid of dog images.
- **Items**: Each card contains an ImageView and a TextView for the breed name.
- **Favorites Bar**: A small horizontal strip or menu access to the 5 favorite dogs.

## 2. Details Screen (Activity or Fragment)
- **Image**: Large-scale view of the selected dog.
- **Metadata**: Displayed breed name and status.
- **Action**: A "Favorite" toggle button.
