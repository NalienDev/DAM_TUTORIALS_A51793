# 02 Features

1. **Random Dog Feed**: Fetch and display a list of random dog images from the Dog CEO API.
2. **Breed Identification**: Automatically parse the image URL to display the breed name as a title.
3. **Pull-to-Refresh**: Refresh the feed to get a new set of random dogs.
4. **Loading States**: Visual feedback (ProgressBar) while images or data are being fetched.
5. **Image Details**: View a single dog image in high resolution with its breed name.
6. **Favorites System**: Save up to 5 favorite dogs (FIFO queue).
7. **Offline Support**: View cached images and favorites without an internet connection.
