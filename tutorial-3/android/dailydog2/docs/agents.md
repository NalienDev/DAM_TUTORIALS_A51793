# AI Agent Guidelines

The AI agent must follow a planning-first approach.

## Strict Rules:

* **Context First:** Always read the documentation inside `/docs` before generating any code.
* **Technology Stack:** Use Kotlin and XML Views ONLY. No Jetpack Compose.
* **Architecture:** Follow MVVM (Model-View-ViewModel).
* **Workflow:** Do not generate large files. Follow the steps in `docs/08\\\_implementation\\\_plan.md` sequentially.
* **Error Handling:** Use professional error handling (try-catch, Result wrappers).
