# 01 Overview - Daily Doggo App

## Purpose

A simple Android application designed to browse random images of dogs from around the world.

## Target Users

Dog lovers who want a quick, visually pleasing way to see different breeds.

## Core System Idea

1. The app fetches a list of random dog images from the Dog CEO API.
2. Users view images in a scrollable grid (RecyclerView).
3. Users can tap an image to see it full-screen with breed details.
4. Users can "Favorite" up to 5 dogs for offline viewing.
